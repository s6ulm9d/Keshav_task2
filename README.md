# Internship Task 2: Nunjucks + Vite Upgrade (Synced with Task 1)

This project is a modularized, production-ready upgrade of **Task 1**. It converts the static HTML structure into a dynamic Nunjucks templating system bundled with Vite.

## 🚀 Key Upgrades (Task 2)

- **🧩 Modular Templating**: Replaced static HTML with Nunjucks components (`layout.njk`, `index.njk`, etc.).
- **⚡ Vite Bundling**: Integrated Vite for ultra-fast development, hot reloading, and optimized production builds.
- **📱 Responsive Evolution**: Enhanced Task 1's styling with custom media queries for full mobile and tablet adaptation.
- **🏗️ Structured Assets**: Organized project assets and templates into a clean `src/` directory.

## 🛠 Technology Stack
- **Templating**: Nunjucks
- **Bundler**: Vite 8.x
- **Framework**: Bootstrap 5 + Custom CSS3
- **Animations**: AOS (Animate On Scroll)
- **Icons**: Font Awesome 6

## 📦 Getting Started

### Installation
```bash
npm install
```

### Run Locally
```bash
npm run dev
```

### Production Build
```bash
npm run build
```
*Compiled files are generated in the `/dist` directory.*
